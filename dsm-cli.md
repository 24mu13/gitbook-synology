# A. DSM CLI

- Check service state (e.g. Web Station):
  ```
  sudo /usr/syno/bin/synosystemctl get-active-status pkg-WebStation*
  ```
  
- Restart service (e.g. Web Station):
  ```
  sudo /usr/syno/bin/synosystemctl restart pkg-WebStation*
  ```
